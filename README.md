Space API Webapp
================

## Usage

```
npm install
grunt
# Open http://localhost:8888/src
```

## Requirements

### Node.js [[+](https://github.com/joyent/node/wiki/Installation)]

### Grunt.js [[+](http://gruntjs.com/getting-started)]
```
$ npm install -g grunt-cli
```
